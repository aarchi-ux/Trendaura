# Trendaura - Clothes Shopping Website

This is an ASP.NET web application for an online clothes shopping platform.

## Features
- Admin Panel
- Add Products
- Add Brands
- Category & Subcategory Management
- User Pages
- Master Page Design

## Technology Used
- ASP.NET
- C#
- SQL Server
- HTML, CSS
